# Third-Party Notices

The AgentOS gateway design is adapted from architectural patterns in
Hermes Agent by Nous Research, licensed under the MIT License:

https://github.com/NousResearch/hermes-agent

Relevant adapted concepts include canonical provider definitions, separation
of provider identity from wire transport, `provider:model` route resolution,
ordered fallback providers, and normalized provider responses. AgentOS code in
this repository is an independent implementation.
