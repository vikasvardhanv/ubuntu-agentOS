# AgentOS Desktop Experience

AgentOS Shell is a GNOME-based session that makes goals, plans, and approvals
first-class while retaining standard Ubuntu applications and accessibility.

## Primary surfaces

- **Global command palette:** text and voice entry for goals, commands, tools,
  applications, and settings; always previews consequential actions.
- **Agent workspace:** active goal, plan graph, artifacts, agent handoffs,
  terminal/browser/file views, and intervention controls.
- **Task timeline:** immutable human-readable action history with evidence,
  costs, duration, policy decisions, and rollback affordances where supported.
- **Approval center:** grouped pending requests showing exact operation,
  resource, risk, reason, data leaving the device, and grant duration.
- **Status area:** runtime health, active agents, model/provider, spend, privacy
  mode, and emergency stop.

Voice input is locally transcribed by default. It never bypasses confirmation
or approval. The shell supports keyboard-only operation, screen readers, and
standard GNOME settings.
